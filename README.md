1. Introduction
Kawatch ("the App") is an emergency assistance application designed to help users send alerts and record evidence during critical situations.
Your privacy and safety are extremely important to us. This Privacy Policy explains how Kawatch handles information when you use the App.
2. Data Collection
Kawatch does not collect, store, transmit, or share personal data on any external servers.
No user accounts are created.
No personal information is uploaded to cloud databases.
No data is sold or shared with third parties.
No analytics or tracking tools are used.
All data processing occurs locally on the user's device.
3. Permissions and Their Purpose
Kawatch requires certain device permissions strictly for emergency functionality:
A. Location Permission
The App accesses device location only when SOS is activated.
Purpose:
To include live location information in emergency SMS alerts.
Location data:
Is not stored on external servers.
Is not tracked continuously.
Is only used during active SOS sessions.
B. SMS Permission
The App uses SMS functionality to:
Send emergency alert messages to preset emergency contacts when SOS is activated.
The App does not:
Send messages without user-triggered SOS activation.
Access or read personal SMS conversations.
C. Microphone Permission
The App uses microphone access only after SOS activation.
Important details:
Audio recording begins only after SOS is activated.
A visible on-screen message informs the user that recording is in progress.
Recording stops immediately when SOS is deactivated.
Audio is NOT automatically uploaded or transmitted.
Audio is saved only if the user manually chooses to send it via MMS.
If the user does not send the recording, it is not stored permanently.
The App does not secretly record audio.
D. Background Services
Kawatch may run limited background services to:
Detect shake gestures for emergency activation.
Ensure SOS functionality works even when the app is not actively open.
These services are used solely for safety-related purposes.
4. Data Storage
All data including:
Emergency contact numbers
SOS session data
Audio recordings (if saved)
App settings
are stored locally on the user's device.
No external servers or cloud databases are used by Kawatch.
5. Data Sharing
Kawatch does not:
Share user data with third parties
Sell user data
Use advertising trackers
Use third-party analytics services
Emergency information is shared only with the emergency contacts selected by the user during SOS activation.
6. User Control
Users have full control over:
Activating or deactivating SOS
Sending or not sending recorded audio
Removing emergency contacts
Uninstalling the App (which removes all locally stored data)
7. Security
Since Kawatch does not use external servers, all data remains under the control of the user on their personal device.
However, users are responsible for maintaining the security of their own device.
8. Children's Privacy
Kawatch does not knowingly collect personal information from children. Since no external data collection occurs, no personal data is stored by the App.
9. Limitation of Liability
Kawatch is a support tool designed to assist during emergency situations.
The App does not guarantee prevention of harm, emergency response, or rescue services. Users acknowledge that Kawatch is an assistance tool and should not replace contacting official emergency services.
